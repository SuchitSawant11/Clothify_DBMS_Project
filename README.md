# 👕 Clothify – Cloth Store Management & Billing System

Clothify is a **web-based cloth store management and billing system** developed as part of the **Database Management Systems (DBMS) Laboratory**.  
The application demonstrates the practical use of **DBMS concepts** through a real-world store scenario using a backend-driven web application.

---

## 🚀 Features
- Product management (add, update, delete products)
- Product image upload and storage
- Customer billing and invoice generation
- Cart-based billing workflow
- Database-backed inventory management
- Clean and responsive UI

---

## 🛠️ Tech Stack
- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS, Tailwind CSS
- **Database:** SQL (Relational Database)
- **Tools & Libraries:** Flask, SQLite / MySQL (as configured)
- **Styling:** Tailwind CSS

---

## 🗄️ DBMS Concepts Used
- Relational database design
- CRUD operations
- Inventory and sales record management
- Primary and foreign key relationships
- Structured data storage and retrieval
